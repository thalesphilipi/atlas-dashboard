Importar Dados

Como funciona a importação

O sistema de importação do Atlas Dashboard permite carregar dados de um arquivo JSON estruturado. Aqui está como funciona:

  • Formato esperado: Arquivo JSON com estrutura de categorias e itens
  • Localização: O arquivo deve estar em "../dados_organizados.json" (pasta pai do projeto)
  • Processamento: O sistema lê o arquivo e organiza os dados em categorias
  • Duplicatas: Itens duplicados são automaticamente detectados e removidos
  • Segurança: Todos os dados são armazenados localmente no SQLite

Estrutura esperada do arquivo JSON:

{
  "categoria1": [
    {
      "title": "Nome do item",
      "username": "usuario@exemplo.com",
      "password": "senha123",
      "url": "https://exemplo.com",
      "notes": "Observações opcionais"
    }
  ],
  "categoria2": [
    // mais itens...
  ]
}

⚠️ Importante: Leia antes de importar

Esta operação irá adicionar os dados do arquivo JSON ao banco atual. Certifique-se de que:

  • Você fez backup dos dados atuais (se necessário)
  • O arquivo "../dados_organizados.json" existe e está no formato correto
  • Não há outros usuários usando o sistema simultaneamente
  • Você tem permissão para importar estes dados
